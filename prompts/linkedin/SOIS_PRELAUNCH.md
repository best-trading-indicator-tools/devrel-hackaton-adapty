# SOIS Pre-launch Playbook

Use this guide for posts that announce the upcoming launch of the State of in-app subscriptions report next week.

Core message framing:
- We have been working on the State of in-app subscriptions report for the past 3 months, and it is almost ready to launch.
- While the content team is adding final touches, share a few screenshots from internal discussions as early insights.
- Include one surprising insight that signals the state of the market.
- Example insight: The top 10% of apps generate 95% of all app revenue.
- Invite participation with a prediction poll and promise later fact-checking with full report data.

Narrative flow:
1. Build anticipation with a launch-timing line (next week).
2. Reveal one teaser insight from the report process.
3. Turn it into a game: ask readers to predict a market benchmark.
4. Offer prizes for correct answers when relevant.
5. Promise to validate predictions after launch using report data.

Tone constraints:
- Sound excited but grounded. No hype words without concrete context.
- Keep copy conversational and human.
- Use direct calls to action for comments or poll participation.
- Never use the acronym "SOIS" in public copy. Write "State of in-app subscriptions report".

Data and evidence:
- This post type is connected to the same report dataset context as SOIS posts.
- Keep all numeric claims grounded in provided evidence.
- If evidence is missing, phrase as a teaser, hypothesis, or open question.
